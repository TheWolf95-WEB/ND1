Проект: Тема WordPress "ndcoder"
Задание: форма обратной связи с выводом отзывов (без плагинов)

1. Установка:
- Установил на ВП (тариф blog на Beget)
- Активировал тему в админке

2. Что реализовано:
- Кастомный тип записи `review` для отзывов
- Форма обратной связи с полями: Имя, Email, Отзыв
- Валидация:
    - Имя — не пустое
    - Email — валидный
    - Отзыв — без HTML и не пустой
- Сообщения об ошибках под каждым полем
- Уведомление об успехе под формой
- POST → REDIRECT → GET (без повторной отправки)
- Вывод последних 3 отзывов снизу
- Стилизация: фон, отступы, кнопка, блоки
- Без плагинов, только через код темы
